      Without using any CSS you can follow the below steps to insert line space in HTML,
      1. Use <br/> tag wherever you want the line space.
      2. Use empty <p></p> tag for single line space. ...
      3. Use &nbsp; as much as you want to make a line space between line, paragraph etc.
      4. Use &nbsp; inside such as <p>, <div>, heading tags etc.
